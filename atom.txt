安装git
安装atom
安装nodejs(选中写入path选项)
进入C://user/当前用户目录/.atom/packages/
当前目录打开gitbash 
先配置镜像
npm config set registry https://registry.npm.taobao.org
然后在atom中搜索该插件的repo 在github中复制地址 然后 
git clone 该地址 
进入clone到的文件夹中 
npm isntall
cd ..
继续下一个插件的操作

常用插件remote-ftp
atom-beautify
atom-braket-highlight
docblockr
emmet
highlight-selected
hyperclick
php-hyperclick
sublime-block-comment
symbols-tree-view
